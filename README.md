# number-guessing-game
A small and interactive game which is built on python for time pass.
## Features
- Picks up a random number from 1 to 100
- Friendly menu interface
- Error handling
## Quick Start

```bash
# 1. Clone or download
git clone https://github.com/sanjan-github/number-guessing-game.git
cd number_guess.py

# 2. Run (Python 3.6+ required)
python number_guess.py
# or
python3 number_guess.py
